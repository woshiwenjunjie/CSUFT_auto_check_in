# 数据库设计（阶段三）

> 技术栈：SQLAlchemy 2.0（ORM）+ PostgreSQL（生产）/ SQLite（开发）
> 迁移工具：Alembic

---

## 1. 设计原则

- **单用户场景优先**（当前 CLI 模式），多用户扩展预留
- 与现有 `cli_config.py` 配置结构兼容，迁移路径平滑
- 所有外键为 `UUID` 主键，不暴露自增 ID
- 时间戳统一用 `UTC` 存储，展示时转 `Asia/Shanghai`
- 敏感字段（password、openid）加密存储，应用层解密

## 3. 表结构

### 3.1 `users` — 用户表

| 列名 | 类型 | 约束 | 说明 |
|------|------|------|------|
| `id` | `UUID` | PK, default `uuid4` | 主键 |
| `username` | `VARCHAR(64)` | UNIQUE, NOT NULL | 学号 |
| `student_id` | `VARCHAR(64)` | | 冗余学号（与 username 同，留作扩展） |
| `password_hash` | `VARCHAR(255)` | | bcrypt 哈希（CLI 界面复用）；空值代表未保存 |
| `openid_encrypted` | `TEXT` | | AES-256-CBC 加密的 OpenID |
| `openid_iv` | `VARCHAR(64)` | | AES IV（base64） |
| `tenant_id` | `VARCHAR(32)` | NOT NULL, default `000000` | 学校租户 ID |
| `token` | `TEXT` | | 最新 access_token（明文存储，服务器时限短） |
| `token_expires_at` | `TIMESTAMP` | | token 过期时间 |
| `task_id` | `VARCHAR(64)` | | 默认任务 ID（CLI 兼容） |
| `is_active` | `BOOLEAN` | default `true` | 是否启用自动打卡 |
| `settings` | `JSONB` | default `{}` | 个性化设置（见 3.1.1） |
| `created_at` | `TIMESTAMP` | NOT NULL, default `now()` | 创建时间 |
| `updated_at` | `TIMESTAMP` | NOT NULL, auto-update | 更新时间 |

#### 3.1.1 `settings` JSONB 结构

```json
{
    "checkin_time": "21:00",

    "random_delay_max": 300,
    "geo_offset_degree": 0.001,
    "force_checkin": false,
    "notify_enabled": true,
    "notify_channel": "serverchan",
    "serverchan_key": ""
}
```

### 3.2 `task_cache` — 任务缓存表

| 列名 | 类型 | 约束 | 说明 |
|------|------|------|------|
| `id` | `UUID` | PK | 主键 |
| `user_id` | `UUID` | FK → `users.id`, NOT NULL | 所属用户 |
| `task_id` | `VARCHAR(64)` | NOT NULL | 学校任务 ID |
| `task_name` | `VARCHAR(255)` | | 任务名称 |
| `scan_type` | `INTEGER` | default `1` | 签到类型 |
| `location_lat` | `NUMERIC(10,7)` | | 宿舍纬度 |
| `location_lng` | `NUMERIC(10,7)` | | 宿舍经度 |
| `location_accuracy` | `NUMERIC(10,2)` | | 打卡范围精度（米） |
| `room_id` | `VARCHAR(64)` | | 宿舍房间 ID |

> UNIQUE(`user_id`, `task_id`) — 每用户每任务一条缓存

### 3.3 `checkin_records` — 打卡记录表

| 列名 | 类型 | 约束 | 说明 |
|------|------|------|------|
| `id` | `UUID` | PK | 主键 |
| `user_id` | `UUID` | FK → `users.id`, NOT NULL | 所属用户 |

## 5. SQLAlchemy 模型

```python
# src/models/db.py — 与现有 __init__.py 共存

import uuid
from datetime import datetime
from sqlalchemy import (
    Column, String, Integer, Numeric, Boolean, Date, Time, Text,
    ForeignKey, JSON, create_engine,
)
from sqlalchemy.dialects.postgresql import UUID, JSONB
from sqlalchemy.orm import DeclarativeBase, Mapped, mapped_column, relationship
from sqlalchemy.sql import func


class Base(DeclarativeBase):
    pass


class User(Base):
    __tablename__ = "users"

    id: Mapped[str] = mapped_column(UUID(as_uuid=False), primary_key=True,
                                     default=lambda: str(uuid.uuid4()))
    username: Mapped[str] = mapped_column(String(64), unique=True, nullable=False)
    student_id: Mapped[str | None] = mapped_column(String(64))
    password_hash: Mapped[str | None] = mapped_column(String(255))
    openid_encrypted: Mapped[str | None] = mapped_column(Text)
    openid_iv: Mapped[str | None] = mapped_column(String(64))
    tenant_id: Mapped[str] = mapped_column(String(32), nullable=False, default="000000")
    token: Mapped[str | None] = mapped_column(Text)
    token_expires_at: Mapped[datetime | None] = mapped_column()
    task_id: Mapped[str | None] = mapped_column(String(64))
    is_active: Mapped[bool] = mapped_column(Boolean, default=True)
    settings: Mapped[dict] = mapped_column(JSONB, default=dict)
    created_at: Mapped[datetime] = mapped_column(server_default=func.now())
    updated_at: Mapped[datetime] = mapped_column(server_default=func.now(),
                                                  onupdate=func.now())

    # relationships
    checkin_records = relationship("CheckinRecord", back_populates="user")
    task_caches = relationship("TaskCache", back_populates="user")
    operation_logs = relationship("OperationLog", back_populates="user")


class TaskCache(Base):
    __tablename__ = "task_cache"

    id: Mapped[str] = mapped_column(UUID(as_uuid=False), primary_key=True,
                                     default=lambda: str(uuid.uuid4()))
    user_id: Mapped[str] = mapped_column(ForeignKey("users.id"), nullable=False)
    task_id: Mapped[str] = mapped_column(String(64), nullable=False)
    task_name: Mapped[str | None] = mapped_column(String(255))
    scan_type: Mapped[int] = mapped_column(Integer, default=1)
    location_lat: Mapped[float | None] = mapped_column(Numeric(10, 7))
    location_lng: Mapped[float | None] = mapped_column(Numeric(10, 7))
    location_accuracy: Mapped[float | None] = mapped_column(Numeric(10, 2))
    room_id: Mapped[str | None] = mapped_column(String(64))
    cached_at: Mapped[datetime] = mapped_column(server_default=func.now())

    __table_args__ = (
        UniqueConstraint("user_id", "task_id", name="uq_task_cache_user_task"),
    )

    user = relationship("User", back_populates="task_caches")


class CheckinRecord(Base):
    __tablename__ = "checkin_records"

    id: Mapped[str] = mapped_column(UUID(as_uuid=False), primary_key=True,
                                     default=lambda: str(uuid.uuid4()))
    user_id: Mapped[str] = mapped_column(ForeignKey("users.id"), nullable=False)
    task_id: Mapped[str] = mapped_column(String(64), nullable=False)
    sign_date: Mapped[datetime.date] = mapped_column(Date, nullable=False)
    sign_time: Mapped[datetime.time | None] = mapped_column(Time)
    sign_lat: Mapped[float | None] = mapped_column(Numeric(10, 7))
    sign_lng: Mapped[float | None] = mapped_column(Numeric(10, 7))
    location_accuracy: Mapped[str | None] = mapped_column(String(32))
    sign_status: Mapped[int | None] = mapped_column(Integer)
    sign_status_name: Mapped[str | None] = mapped_column(String(32))
    is_late: Mapped[bool] = mapped_column(Boolean, default=False)
    is_auto: Mapped[bool] = mapped_column(Boolean, default=True)
    raw_response: Mapped[dict | None] = mapped_column(JSONB)
    error_message: Mapped[str | None] = mapped_column(Text)
    created_at: Mapped[datetime] = mapped_column(server_default=func.now())

    __table_args__ = (
        UniqueConstraint("user_id", "task_id", "sign_date",
                         name="uq_record_user_task_date"),
    )

    user = relationship("User", back_populates="checkin_records")


class OperationLog(Base):
    __tablename__ = "operation_logs"

    id: Mapped[str] = mapped_column(UUID(as_uuid=False), primary_key=True,
                                     default=lambda: str(uuid.uuid4()))
    user_id: Mapped[str | None] = mapped_column(ForeignKey("users.id"))
    level: Mapped[str] = mapped_column(String(16), nullable=False, default="INFO")
    action: Mapped[str] = mapped_column(String(64), nullable=False)
    message: Mapped[str | None] = mapped_column(Text)
    detail: Mapped[dict | None] = mapped_column(JSONB)
    ip_address: Mapped[str | None] = mapped_column(String(45))
    created_at: Mapped[datetime] = mapped_column(server_default=func.now())

    user = relationship("User", back_populates="operation_logs")


class GlobalConfig(Base):
    __tablename__ = "configs"

    id: Mapped[str] = mapped_column(UUID(as_uuid=False), primary_key=True,
                                     default=lambda: str(uuid.uuid4()))
    key: Mapped[str] = mapped_column(String(128), unique=True, nullable=False)
    value: Mapped[str | None] = mapped_column(Text)
    description: Mapped[str | None] = mapped_column(String(255))
    updated_at: Mapped[datetime] = mapped_column(server_default=func.now(),
                                                  onupdate=func.now())
```

## 6. 数据流

### 6.1 登录流程

```
CLI/API 收到 OpenID + 学号
    │
    ▼
查询 users (username) → 无则创建 User 记录
    │
    ▼
调用学校 OAuth → 获取 access_token
    │
    ▼
更新 User.token / token_expires_at
    │
    ▼
返回 token → 客户端保存（CLI 写 config.json / API 返回 JWT）
```

### 6.2 打卡流程

```
定时器触发 / 手动触发
    │
    ▼
查 users (is_active=true) → 遍历活跃用户
    │
    ▼
查 task_cache (user_id, task_id)
  └─ 无缓存 / 缓存过期 → 调 getTaskByIdForApp → 写入 task_cache
    │
    ▼
生成模拟坐标 → 调 stuSign / stuLateSign
    │
    ▼
写入 checkin_records + operation_logs
    │
    ▼
ServerChan 通知（可选）
```

### 6.3 月度查询

```
用户请求 /records?month=2026-06
    │
    ▼
SELECT * FROM checkin_records
WHERE user_id = ? AND sign_date BETWEEN '2026-06-01' AND '2026-06-30'
ORDER BY sign_date
```

## 7. 安全设计

| 字段 | 策略 | 说明 |
|------|------|------|
| `password_hash` | bcrypt (`werkzeug.security`) | 仅用于 CLI 密码复存；API 登录对接学校 MD5+OAuth |
| `openid_encrypted` | AES-256-CBC + 随机 IV | 密钥存环境变量 `OPENID_ENCRYPT_KEY` |
| `token` | 明文存储 | 学校 token 有效期短（约 2 小时），泄露影响小 |
| API 鉴权 | JWT（FastAPI + python-jose） | 管理接口用 JWT，打卡接口用 school token |

## 8. 兼容与迁移

### 8.1 从 CLI config.json 迁移

```python
def migrate_cli_config_to_db():
    """一次性导入 CLI 配置到数据库"""
    cfg = load_config()
    if not cfg:
        return
    user = User(
        username=cfg["username"],
        password_hash=hash_password(get_password(cfg)),
        openid_encrypted=encrypt_openid(cfg.get("openid", "")),
        tenant_id=cfg.get("tenant_id", "000000"),
        token=cfg.get("token", ""),
        task_id=cfg.get("task_id", ""),
    )
    session.add(user)
    session.commit()
```

### 8.2 数据库连接

```python
# src/core/database.py
from sqlalchemy import create_engine
from sqlalchemy.orm import sessionmaker

DATABASE_URL = os.getenv("DATABASE_URL", "sqlite:///~/.auto_check_in/data.db")

engine = create_engine(DATABASE_URL)
SessionLocal = sessionmaker(bind=engine)


def get_db():
    db = SessionLocal()
    try:
        yield db
    finally:
        db.close()
```

建议开发期用 SQLite（零配置），生产用 PostgreSQL（连接池 / 并发 / JSONB）。
