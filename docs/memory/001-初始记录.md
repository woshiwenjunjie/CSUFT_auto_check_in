# 项目记忆 — 初始记录

> 创建时间：2026-06-07

---

## 项目目标
复刻 nahs.online 文章的自动晚点名打卡系统，适用于学校微信小程序打卡。

## 技术决策
- **后端框架**：FastAPI（与文章一致）
- **任务调度**：APScheduler（与文章一致）
- **数据库**：开发期用 SQLite，上线切换 PostgreSQL
- **部署**：Windows 开发，Linux 服务器部署

## 已知约束
- 学校打卡系统使用飞源（flySource）统一认证平台
- 请求签名算法为双层 MD5，请求头为 `FlySource-sign`
- OpenID 必须从客户端抓包获取，后端无法直接获取
- stuTaskId 为客户端按固定规则 MD5 生成，非服务端返回

## 已准备的环境
- Python 3.14.5 虚拟环境 `.venv/`
- FastAPI + APScheduler + SQLAlchemy + 加密库已安装
- wxapp-unpacker（npm 全局）可用于反编译小程序
- Git 仓库已初始化
