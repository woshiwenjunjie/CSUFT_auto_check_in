# 008 — CLI 用户体验全面升级

日期：2026-06-07

## 改动概要

v0.7.1 对 CLI 工具做了全面的用户体验升级：从 7 个命令扩展到 10 个，新增终端彩色输出、API 调用动画指示器、交互式配置向导、状态概览命令，以及引导式错误恢复。

零新增依赖，纯 Python 标准库实现。

## 新增命令

### `setup` — 交互式首次配置向导
- Step 1: 引导用户获取并输入 OpenID
- Step 2: 输入学号
- Step 3: 选择是否保存密码
- Step 4: 自动测试登录并保存配置
- 登录失败时仍保存凭据，提示 `login-openid` 重试

### `status` — 一键状态概览
- 显示配置文件路径、学号、OpenID（掩码）、密码状态
- Token 有效性实时校验（调用 tasks API）
- 当前任务名称 + 打卡时间窗口
- 今日打卡记录（日期/状态/坐标）
- 未登录或 Token 过期给出明确的修复命令

### `config` — 配置管理
- `config show` — 所有字段掩码显示（敏感信息仅显示首尾 4 字符）
- `config clear` — 清除 token（默认）
- `config clear --all` — 清除全部（需二次确认）
- `config clear --password` — 仅清除密码

## 终端样式系统

```python
class Style:  # ANSI 转义码
    success(text) → 绿色    error(text) → 红色
    warning(text) → 黄色    info(text) → 青色
    muted(text) → 灰色      bold(text) → 粗体
    heading(text) → 粗体+青色
```

- `c(style_fn, text)` — 仅在 `USE_COLOR=True` 时应用样式
- `USE_COLOR` = `NO_COLOR` 未设置 AND `isatty()` — 管道/重定向时自动降级
- `Spinner` — 线程动画（braille `⠋⠙⠹⠸⠼⠴⠦⠧⠇⠏`），TTY 关闭时自动跳过
- `divider(title)` — 分段线 + 可选标题
- `kv(key, value)` — 对齐键值行
- `bullet(text, ok)` — `✓`/`✗` 图标列表

## 输出格式化改进

| 命令 | 改进 |
|------|------|
| `tasks` | 编号列表 `[1]` + 任务名高亮青色 + 分段标题 |
| `detail` | 分"基本信息"和"宿舍信息"两部分，定位校验状态着色 |
| `checkin` | 提交前显示完整参数面板（坐标/距离/精度），范围外给出具体阈值 |
| `record` | 状态按类型着色（正常=绿 / 迟到=黄 / 其他=灰） |
| `month` | ✓=正常 / !=迟到 / −=其他 + 底部汇总统计 |

## 错误恢复引导

每种常见错误都会打印具体修复命令：

- Token 过期 → `python scripts/cli.py login-openid`
- 绑定冲突 → `--bind 0`
- 密码错误 → `--force-input`
- 缺少 task_id → 先运行 `tasks`
- 超出打卡范围 → `--force` 或调整 `--offset`

## 编码修复

Windows GBK 终端（bash/cmd）中 Python 可能从控制台代码页继承 GBK 编码，导致中文乱码。脚本启动时检测并自动修复：

```python
if sys.platform == "win32" and hasattr(sys.stdout, "reconfigure"):
    sys.stdout.reconfigure(encoding="utf-8", errors="replace")
```
