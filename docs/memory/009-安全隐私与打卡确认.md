# 记忆 — v0.7.2 安全隐私提升与打卡确认

日期：2026-06-08

## 变更摘要

### 学号脱敏
CLI 中 3 处学号明文显示改为 `_mask(username, 3)` 掩码格式：
- `cmd_status`（状态概览）
- `cmd_config show`（配置查看）
- `cmd_login_openid` 登录成功输出

掩码规则：前 3 位 + `***...***` + 后 3 位（如 `202****101`）。≤6 位学号全掩。

**Why:** 终端截图/录屏可能泄露学号。学号是 PII 敏感信息，应与 OpenID/密码/Token 同等对待。

### 打卡确认查询
`cmd_checkin` 提交打卡后不再仅信任 API 返回的 success 消息，而是额外调用 `get_one_record` 回查服务器确认，展示日期/状态（颜色编码）/坐标/打卡时间。

**Why:** 服务器可能返回 success 但实际记录状态不同（如"迟到"），信任但验证。

### 中文回复规范
CLAUDE.md 和 AGENTS.md 新增"所有最终回复必须用中文"约定。代码/命令/标识符保持英文。
