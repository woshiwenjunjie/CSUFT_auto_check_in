# 004 — CLI 工具开发完成

日期：2026-06-07

## 阶段二完成内容

### 核心模块

#### `src/utils/crypto.py` — 加密工具
- `md5(data: str) -> str`：MD5 哈希
- `b64_encode(data: bytes) -> str`：Base64 编码
- `b64_decode(data: str) -> bytes`：Base64 解码

#### `src/utils/sign.py` — 签名算法
- `generate_sign(path, timestamp, token)`：生成 `FlySource-sign` 请求头值
  ```
  sign = MD5(path + "?sign=" + MD5(str(ts) + token)) + "1." + base64(str(ts))
  ```
- `generate_basic_auth()`：生成 Basic Auth 头（Base64(ClientId:ClientSecret)）
- 凭据通过环境变量 `FLYSOURCE_CLIENT_ID` / `FLYSOURCE_CLIENT_SECRET` 读取，支持覆盖

#### `src/utils/geo.py` — 定位模块
- `haversine(lat1, lng1, lat2, lng2) -> float`：球面距离（米）
- `random_offset(lat, lng, max_offset=0.0005) -> (lat, lng)`：GPS 坐标随机偏移

#### `src/core/client.py` — API 客户端
- 封装 httpx，统一错误处理
- 单连接池复用（`self._client`）
- 8 个接口：get_captcha、sign_in、sign_in_openid、get_task_list、get_task_detail、get_one_record、stu_sign、stu_late_sign、get_month_records

### CLI 工具 `scripts/cli.py`

| 子命令 | 参数 | 功能 |
|--------|------|------|
| login | username, [password] | 密码登录，密码留空则交互式输入 |
| login-openid | openid, [username], [password] | OpenID 绑定微信登录 |
| tasks | --page, --size | 分页查询任务列表 |
| detail | task_id | 查看任务详情（含坐标、精度） |
| checkin | task_id, --lat/lng/offset/force/late-date/file-id | 提交打卡/补签 |
| record | task_id, --date | 查询打卡状态 |
| month | task_id, month | 按月查询打卡记录 |

配置自动持久化到 `~/.auto_check_in/config.json`。

## 安全处理
- CLIENT_ID/CLIENT_SECRET 不再硬编码，优先环境变量
