### 2.5 `scripts/cli.py` (256 行)

| 项目 | 状态 |
|------|------|
| 参数解析 | ✅ argparse 使用规范 |
| 密码安全 | ✅ getpass 交互式输入 |
| 配置持久化 | ✅ `~/.auto_check_in/config.json` |
| 功能完整性 | ✅ 7 个子命令覆盖全部需求 |

**重要发现：**

1. **`cmd_login_openid` 密码三目表达式**（第 49 行）：
   ```python
   password = args.password or getpass.getpass("密码: ") if args.password is None else args.password
   ```
   在 Python 中解析为 `(args.password or getpass... ) if args.password is None else args.password`。
   当 `args.password is None` 时，`None or getpass(...)` 触发 getpass，结果正确。但可读性差，建议改为：
   ```python
   if args.password is None:
       password = getpass.getpass("密码: ")
   else:
       password = args.password
   ```

2. **配置双重加载**（第 37-38 行）：`cmd_login` 调用 `get_client()` 加载一次 config，
   然后 `load_config()` 再次加载。非 bug，但可复用 `cfg` 返回值。

### 4.2 待改进项

5. **FastAPI 集成准备**：添加 CORS、lifespan 事件、全局异常处理器
