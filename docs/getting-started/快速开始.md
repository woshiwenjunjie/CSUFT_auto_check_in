# 快速开始

> 5 分钟完成首次配置，实现一键打卡（v0.11.0 — 支持多账号批量打卡）

## 前置要求

- Python 3.10+
- Node.js（可选，用于签名交叉验证）
- mitmproxy（可选，用于自动捕获 OpenID）

## 1. 安装依赖

```powershell
# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境
.\.venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

## 2. 获取 OpenID

OpenID 是微信小程序用户的唯一标识，必须通过抓包获取。

**⭐ 推荐方式：模拟器一键自动捕获**

```powershell
pip install mitmproxy
python scripts/tools/capture_openid_emulator.py --port 8899
```

脚本自动完成证书安装 + 代理配置 + mitmproxy 启动 + OpenID 提取。详见 [模拟器自动抓取 OpenID](../guides/user/模拟器自动抓取OpenID.md)。

**备用方式：**

- **Reqable**（无需 root，手机单机，热点可用）→ [认证流程与抓包分析](../reference/认证流程与抓包分析.md)
- **Fiddler**（PC + 模拟器/手机，需 root）→ [Fiddler 抓包指南](../guides/user/fiddler-抓包获取OpenID.md)

## 3. 交互式配置

```powershell
python scripts/cli.py setup
```

按提示依次输入：OpenID → 学号 → 密码，系统自动验证登录并保存配置。

## 4. 一键打卡

```powershell
# 查看状态
python scripts/cli.py status

# 打卡签到（默认全部已配置账号）
python scripts/cli.py checkin
```

## 多账号管理

```powershell
# 查看已有账号
python scripts/cli.py config profile list

# 切换账号
python scripts/cli.py config profile USER_2

# 批量打卡
python scripts/cli.py checkin --profiles USER_1,USER_2,USER_3
```
