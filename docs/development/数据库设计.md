# 数据库设计（阶段三）

> 技术栈：SQLAlchemy 2.0（ORM）+ PostgreSQL（生产）/ SQLite（开发）
> 迁移工具：Alembic

## 设计原则

- 单用户场景优先（当前 CLI 模式），多用户扩展预留
- 与现有 `cli_config.py` 配置结构兼容，迁移路径平滑
- 所有外键为 UUID 主键，不暴露自增 ID
- 时间戳统一用 UTC 存储，展示时转 `Asia/Shanghai`
- 敏感字段（password、openid）加密存储，应用层解密

## ER 图

```
users (1)────(1) tokens             每个用户一个活跃 token
```

## 索引设计

```sql
-- users
CREATE UNIQUE INDEX idx_users_username ON users(student_id);
CREATE INDEX idx_users_is_active ON users(is_active) WHERE is_active = TRUE;

-- tokens
CREATE UNIQUE INDEX idx_tokens_user ON tokens(user_id);

-- task_cache
CREATE UNIQUE INDEX idx_task_cache_user_task ON task_cache(user_id, task_id);

-- checkin_records
CREATE UNIQUE INDEX idx_records_user_task_date ON checkin_records(user_id, task_id, sign_date);
CREATE INDEX idx_records_user_date ON checkin_records(user_id, sign_date);
CREATE INDEX idx_records_status ON checkin_records(sign_status);

-- operation_logs
CREATE INDEX idx_op_logs_user ON operation_logs(user_id, created_at);
CREATE INDEX idx_op_logs_level ON operation_logs(level, created_at);

-- schedules
CREATE UNIQUE INDEX idx_schedules_user ON schedules(user_id);
```

## 迁移策略

1. 安装 SQLAlchemy + Alembic + psycopg2
2. 创建 `src/models/db.py`（ORM 模型）
3. 创建 `src/core/database.py`（引擎 + 会话）
4. 配置 Alembic（`alembic init`）
5. `alembic revision --autogenerate -m "init"` 生成初始迁移
6. 实现 CLI→DB 迁移脚本 `scripts/migrate_config.py`

## CLI config.json 迁移

```python
def migrate_cli_config_to_db():
    """一次性导入 CLI 配置到数据库"""
    cfg = load_config()
    if not cfg:
        return
    user = User(
        username=cfg["username"],
        password_hash=hash_password(get_password(cfg)),
        openid_encrypted=encrypt_openid(cfg.get("openid", "")),
        tenant_id=cfg.get("tenant_id", "000000"),
        token=cfg.get("token", ""),
        task_id=cfg.get("task_id", ""),
    )
    session.add(user)
    session.commit()
```

## 安全设计

| 字段 | 策略 | 说明 |
|------|------|------|
| `password_enc` | AES-256-CBC + 随机 IV | 密钥存环境变量 `ENCRYPTION_KEY` |
| `openid_encrypted` | AES-256-CBC + 随机 IV | 密钥存环境变量 |
| `token` | 明文存储 | 学校 token 有效期短，泄露影响小 |
| API 鉴权 | JWT（FastAPI + python-jose） | 管理接口用 JWT |
