# 添加新账号教程

> 适用版本：v2.0.0+
> 功能：为本工具添加第二个、第三个……账号，支持多用户批量打卡

---

## 1. 账号是什么？

本工具用 **profile** 管理多账号。每个 profile 是一个独立账号，包含：

- **OpenID** — 微信小程序身份标识（o 开头，约 28 位）
- **学号** — 你的学号
- **Token** — 登录后获取的访问凭证

profile 名为 `default` / `USER_2` / `USER_3` … 可通过 `--profile` 指定。

> **设计说明**：密码字段可用 `--save-password` 保存（`$obf:` 混淆，仅防肩窥，非加密）。

---

## 2. 添加新账号（3 步完成）

### 2.1 抓取 OpenID → 自动登录（一步完成）

capture-openid 捕获 OpenID 后**自动调用登录 API 获取 token**，全程无需额外操作。

```bash
# 推荐：模拟器自动抓取
python scripts/tools/capture_openid_emulator.py --profile USER_N

# 或：mitmproxy 自动抓取 + 登录
python scripts/cli.py capture-openid --profile USER_N
```

> 已有 OpenID 可跳过此步。查看 `~/.auto_check_in/config.json` 确认。

捕获完成后 profile 包含完整凭据，可直接打卡。

### 2.2 验证

```bash
# 查看今日任务
python scripts/cli.py tasks --profile USER_N

# 批量打卡测试（推荐：全部账号一起打卡）
python scripts/cli.py checkin

# 或仅打卡新账号
python scripts/cli.py checkin --profiles USER_N
```

### 2.3 SCF 同步

如果使用了腾讯云 SCF 自动打卡，登录后需同步配置并重新部署：

```bash
# 同步配置到 password.txt 和 SCF 环境变量
python scripts/cli.py config sync

# 重新生成 SCF 部署包（自动包含新账号信息）
cd deploy/tencent-scf
python deploy.py gen-env
python deploy.py --env-json scf_env.json
```

## 3. 管理多账号

### 切换当前账号

```bash
python scripts/cli.py config profile USER_2
```

### 查看所有账号

```bash
python scripts/cli.py config profile list
```

### 查看某个账号的配置

```bash
python scripts/cli.py config --profile USER_2
```

### 批量打卡

```bash
# 打卡全部已配置的账号
python scripts/cli.py checkin

# 打卡指定账号
python scripts/cli.py checkin --profiles USER_1,USER_2,USER_3
```

## 4. 完整示例：添加 USER_4

```bash
# 1. 抓取 + 自动登录（一步完成）
python scripts/tools/capture_openid_emulator.py --profile USER_4

# 2. 验证
python scripts/cli.py checkin --profiles USER_4
```

## 常见问题

### Q: 如何为已有用户更新 OpenID？
A: 重新运行 `capture-openid --profile USER_N` 即可。新 OpenID 自动覆盖旧值，并重新获取 token。

### Q: 密码保存了但登录失败？
A: 用 `--force-input` 重新输入密码：
```bash
python scripts/cli.py login-openid --profile USER_N --username 学号 --force-input
```

### Q: config sync 命令还能手动用吗？
A: 可以，任何时候想手动同步都行：
```bash
python scripts/cli.py config sync
```
